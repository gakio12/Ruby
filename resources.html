Installing Ruby

Ruby is already installed on Mac OSX and most Linux distributions. However, these pre-installed versions may not be the most up to date.

  • Mac OSX: One common way to update Ruby is to use Homebrew and simply type:

    • $ brew install ruby

  • Linux: Installation is as simple as whatever your distribution uses to install packages. More info can be found at the provided link.

  • Windows: There is an awesome Ruby installer just for Windows! It can be found here.

    • Additional installation instructions (or distribution specific instructions) can be found here: https://www.ruby-lang.org/en/documentation/installation/.
