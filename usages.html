Usages

One of Ruby's main usages is for using it with the Rails framework. This framework makes making web applications easy and intuitive. Ruby on Rails is used similarly in the way PHP is used for web applications. Ruby itself, however, is not like PHP. Rails provides default structures for a databse, a web application, and web pages.

Ruby itself can be used for a variety of applications. It is a general scripting language so, for example, you can write a simple text file parser with regular expressions. Ruby itself is more similar, in application, to Python and Perl.
