Usages

One of Ruby's main usages is for using it with the Rails framework. This framework helps making web applications easy and intuitive. Rails provides default structures for a databse, a web application, and web pages.

Some examples of websites that were built with Ruby on Rails include:

  • GitHub
  • Airbnb
  • Twitch
  • SoundCloud
  • Hulu

Ruby itself can be used for a variety of applications. It is a general scripting language so, for example, you can write a simple text file parser with regular expressions. Ruby itself is more similar, in application, to Python and Perl.

From ruby’s website, they describe many other applications that are written in Ruby:

Simulations:

  • NASA Langley Research Center in Virginia uses ruby to conduct simulations.
  • A research group in Motorola uses Ruby to script a simulator.

3D Modeling:

  • Google SketchUp is a 3D modeling application that uses Ruby for its macro scripting API.

Networking

  • Open Domain Server uses Ruby to allow people using dynamic DNS clients to udpate, in real time, their IP configuration so that it can be mapped to static domains.
